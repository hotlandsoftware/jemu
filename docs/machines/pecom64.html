The PECOM 64 is a Yugoslavic computer released in 1987. It is the successor to the PECOM 32. Essentially the same as the PECOM 32 but a bigger ROM size (32KB vs 16KB).

Run
./bin/jemu-rca -M pecom64 -rom /path/to/roms/
