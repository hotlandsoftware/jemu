An arcade game produced in 1981 by Cidelsa, a Spanish company. It is the sequel to Altair. Shoot-em-up game with similar gameplay to Destroyer, except you can move up and down and lose parts of your spaceship.

This machine supports dip switches. You can use dipswitch list in the GEMU console for a list and dipswitch (name) (value) to set the value.

Run
./bin/gemu-rca -M altair2 -rom /path/to/roms/altair2/
